Como usar esta planilha

Protocolo: cada participante preenche as colunas Ambig., Escopo, Risco e Classe individualmente e sem mostrar para os outros. Depois compare — as divergências são o material de ouro. A coluna Justificativa é obrigatória: sem ela, não dá para aprender com os desacordos.
Régua rápida: Ambig. A1=especificado · A2=solução em aberto · A3=caminho incerto · A4=sem definição prévia · A5=define princípios. Escopo 1=rotina · 2=módulo · 3=multi-módulo · 4=produto · 5=empresa. Risco: S (sim) se houver regulatório / irreversível / dados sensíveis / produção crítica; N caso contrário. Classe = A-base + (+1 se Escopo≥3 ou Risco=S). Ver Guia de Classificação para detalhes.
Os dados preenchidos são exportados para uso em planilha.
